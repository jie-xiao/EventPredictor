### T3.2 信息收集 Agent
- 任务：收集事件的基本信息
- 输入：Event
- 输出：CollectedInfo
- Prompt要点：
  - 提取事件关键信息
  - 关联背景信息
  - 识别信息来源可靠性

### T3.3 深度分析 Agent
- 任务：分析事件影响
- 输入：CollectedInfo
- 输出：Analysis
- Prompt要点：
  - 评估影响范围
  - 判断持续时间
  - 识别关键因素
  - 分析市场情绪

### T3.4 趋势预测 Agent
- 任务：生成预测
- 输入：Analysis
- 输出：Prediction
- Prompt要点：
  - 综合分析结果
  - 给出趋势判断
  - 提供置信度
  - 列出影响因素
