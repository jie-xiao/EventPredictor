# EventPredictor 前端设计规范

## 一、设计风格定位

**风格**：专业情报/金融交易系统风格
**关键词**：高端、专业、深色主题、数据可视化
**参考**：Bloomberg Terminal、Palantir Foundry、核心理念

## 六、组件设计规范

### 6.1 Header 组件
```
┌─────────────────────────────────────────────────────────────────────────┐
│ [Logo] Global Situation Prediction  |  20 Events | 3 Critical | [Clock] [Refresh] │
│     AUTOMATIC ANALYSIS DASHBOARD    |                                        │
└─────────────────────────────────────────────────────────────────────────┘
```
- 背景：`rgba(10, 14, 26, 0.9)` + backdrop-blur
- 高度：68px
- 分割线：1px solid rgba(30, 41, 59, 0.5)

### 6.2 Globe Section
- 3D地球占60%宽度
- 事件标记：彩色圆点，脉冲动画
- 国家高亮：热力图效果
- 交互：点击标记显示详情

### 6.3 Prediction Panel
```

│                             │
├─────────────────────────────┤
│  RISK: High                 │
│  CONFLICTS: 2               │
├─────────────────────────────┤
│  📌 Key Insights             │

- 趋势图标：大尺寸，动态颜色
- 置信度：进度条+数字

- 按钮：渐变背景，发光效果

### 6.5 Event Card
```
┌──────────────────────────┐
│ 🔴 HIGH                  │  <- 严重度标记

- 边框：1px solid rgba(30, 41, 59, 0.5)

## 七、动效规范

### 7.1 过渡动画
- 所有过渡：`transition: all 0.2s ease`
- 悬停抬起：`transform: translateY(-2px)`
- 点击反馈：`transform: scale(0.98)`

### 7.2 加载动画
```css
/* 脉冲发光 */
@keyframes pulse-glow {
  0%, 100% { box-shadow: 0 0 20px rgba(0, 229, 255, 0.3); }
  50% { box-shadow: 0 0 40px rgba(0, 229, 255, 0.6); }
}

/* 骨架屏 */
@keyframes shimmer {
  0% { background-position: -200% 0; }
  100% { background-position: 200% 0; }
}
```

### 7.3 数据更新
- 新数据进入：淡入 + 轻微上移
- 数字变化：计数动画
- 状态变化：颜色渐变

## 八、图标规范

使用 Lucide React 图标库：
- 趋势：`TrendingUp`, `TrendingDown`, `Minus`
- 风险：`Shield`, `AlertTriangle`
- 时间：`Clock`
- 位置：`MapPin`
- 操作：`RefreshCw`, `ChevronRight`
- 状态：`Activity`, `Zap`, `Globe`

## 九、数据展示规范

### 数字
- 大数字：32px, 粗体, 主色
- 变化：绿色+/红色-
- 格式：千分位分隔

### 状态标签
- 高危：红色背景 `rgba(239, 68, 68, 0.2)`
- 中等：橙色背景 `rgba(245, 158, 11, 0.2)`
- 低风险：绿色背景 `rgba(16, 185, 129, 0.2)`

### 进度条
- 背景：`rgba(30, 41, 59, 0.5)`
- 填充：渐变色
- 圆角：full
