# AnalysisResult Page Design Specification

## Overview

The AnalysisResult page is the detailed view for multi-agent event analysis. This design transforms the current tabbed layout into a professional, high-density dashboard that matches the Bloomberg Terminal/Palantir Foundry aesthetic established on the homepage.

---

## Design Philosophy

- **Information Density**: Maximum useful information per screen area
- **Visual Hierarchy**: Clear scanning paths with progressive disclosure
- **Action-Oriented**: Key insights and predictions always visible
- **Professional Aesthetic**: Dark theme with cyan/blue accents, subtle glows, technical precision

## Color System Application

### Primary Accents
- `#00E5FF` (Primary Cyan): Key highlights, active states, trend indicators
- `#0077FF` (Primary Blue): Secondary highlights, interactive elements

### Status Colors
- `#10B981` (Success): Positive trends, low risk, consensus, synergies
- `#F59E0B` (Warning): Medium risk, caution, sideways trends
- `#EF4444` (Danger): Negative trends, high risk, conflicts
- `#3B82F6` (Info): Neutral information, timeline markers

### Backgrounds
- `#0A0E1A`: Page background
- `#0F172A`: Card background (default)
- `#0F172A` with alpha variations: Nested cards, sections
- `#1E293B`: Borders, dividers (with 50% opacity for subtle)

### Special Effects
- Glow: `0 0 30px rgba(0, 229, 255, 0.3)` for key elements
- Gradient headers: `linear-gradient(90deg, rgba(0,229,255,0.1) 0%, transparent 100%)`

## Animation & Transitions

### Page Load
1. Header slides down (200ms)
2. Event Summary fades in left-to-right (300ms, delay 100ms)
3. Prediction Panel fades in right-to-left (300ms, delay 150ms)
4. Role Matrix slides up (250ms, delay 200ms)
5. Cross-Analysis fades in (300ms, delay 250ms)

### Section Transitions
- Collapse: Height animates to 0, opacity fades (200ms ease)
- Expand: Height animates from 0, opacity fades in (300ms ease-out)
- Hover: Scale 1.01, subtle glow (200ms)

### Data Updates
- Confidence: Counter animation (number tween)
- Progress bars: Width transition (500ms ease-out)
- Trend change: Icon rotation + color transition (300ms)

## Accessibility

### Keyboard Navigation
- Tab: Navigate through focusable elements
- Enter/Space: Activate buttons, expand sections
- Arrow keys: Navigate lists, grids
- Escape: Close modals, return to overview

### Screen Readers
- ARIA labels on all interactive elements
- Live regions for dynamic content updates
- Semantic HTML structure
- Alt text for all visualizations

### Visual Accessibility
- High contrast ratios (WCAG AA minimum)
- Focus indicators on all focusable elements
- Text resize support (100-200%)
- Color alternatives (icons, patterns)

### Rendering
- Virtual scrolling for long lists

- Code splitting for heavy components

- Efficient data structures for large datasets
- Web Workers for intensive computations
- Debounced search/filter operations
- Caching for frequently accessed data
